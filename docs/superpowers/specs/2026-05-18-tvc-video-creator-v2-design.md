# TVC广告视频创意生成器 V2 - 设计文档

## 基本信息

- **Skill名称：** `tvc-video-creator-v2`
- **创建日期：** 2026-05-18
- **版本：** 2.0
- **状态：** 设计完成

## 核心定位

面向面料/服装行业的TVC广告视频创意生成器，专注于30秒竖屏（9:16）品牌大片级视频的创意构思、脚本撰写和分镜制作。

## 核心原则

1. **面料质感优先** - 所有创意、视觉风格、脚本都要围绕"让面料看起来高级、服装好看"展开
2. **品牌大片标准** - 不是电商流水线，要有故事性、艺术感、电影质感
3. **一致性第一** - 服装与参考图100%一致，面料纹理细节必须正确
4. **场景适配** - 场景选择要与服装风格、品牌调性、目标受众高度匹配

## 输入输出

### 输入素材
- 面料细节图（用户上传）
- 服装图（不同类型的：内衣、内裤、家居服、童装等）

### 输出成果
- 两张15秒对应的九宫格分镜图（纯文本格式）
- 总时长30秒（两段各15秒）

### 技术适配
- 生图模型：GPT-image2（英文提示词 + 中文描述）
- 视频模型：Seedance2.0（中文提示词）

## 流程设计

```
输入素材（面料图+服装图）
    ↓
模块1：创意构思（3组方向）→ 用户选择1组
    ↓
模块2：视觉风格（3种选择）→ 用户选择1种
    ↓
模块3：概念图生成（2张描述）→ 用户确认调性
    ↓
模块4：脚本撰写（3版脚本）→ 用户选择1版
    ↓
模块5：分镜制作（两张15秒九宫格分镜图）
    ↓
最终交付
```

## 模块详细设计

### 模块1：创意构思模块

**功能：** 以顶尖TVC创意大师的视角，根据面料特点和服装风格，构思整体创意

**输入：** 面料图 + 服装图

**输出：** 3组创意方向，每组包含：
- 创意主题/概念
- 核心视觉表现
- 情感调性
- 目标受众感受
- 场景选择理由

**关键要求：**
- 突出面料的高级质感
- 场景选择要与服装风格匹配
- 品牌大片标准，不是电商流水线

**示例输出格式：**
```
创意方向A：
主题：XXX
视觉表现：XXX
情感调性：XXX
目标受众感受：XXX
场景选择：XXX

创意方向B：
...

创意方向C：
...
```

### 模块2：视觉风格模块

**功能：** 根据选定的创意方向，提供3种不同的视觉风格供用户选择

**输入：** 用户选定的创意方向

**输出：** 3种视觉风格选择，每种包含：
- 风格名称（如：极简主义、电影质感、梦幻柔光等）
- 画面调性描述
- 色彩倾向
- 光景特点
- 参考风格关键词

**关键要求：**
- 风格要能突出面料质感
- 品牌大片标准

**示例输出格式：**
```
风格A：极简主义
画面调性：干净、留白、高级感
色彩倾向：低饱和度、莫兰迪色系
光影特点：柔和自然光

风格B：电影质感
...

风格C：梦幻柔光
...
```

### 模块3：概念图生成模块

**功能：** 根据选定的创意方向和视觉风格，生成2张概念图的描述（适配GPT-image2模型）

**输入：** 选定的创意方向 + 视觉风格

**输出：** 2张概念图的详细描述，每张包含：
- 画面构图描述
- 色彩/光影描述
- 主体（模特+服装）呈现方式
- 背景/环境描述
- 适配GPT-image2的英文提示词
- 中文描述（方便用户确认）

**关键要求：**
- 确保服装与参考图100%一致
- 面料纹理细节必须正确
- 英文提示词 + 中文描述

**示例输出格式：**
```
概念图A：
画面描述：XXX
中文描述：XXX
GPT-image2英文提示词：[针对该模型优化的英文提示词]

概念图B：
...
```

### 模块4：脚本撰写模块

**功能：** 根据选定的创意方向、视觉风格和确认的概念图调性，撰写视频脚本

**输入：** 选定的创意方向 + 视觉风格 + 概念图调性 + 面料特点 + 服装风格

**输出：** 3版脚本供用户挑选，每版包含：
- 视频总时长：30秒（两段各15秒）
- **视觉调性**（与面料质感和服装风格相匹配）
- **景别**（特写/近景/中景/远景，必须多样化）
- **摄像机运动**
- **镜头类型**
- **焦距**
- **光圈**
- **画面内容描述**
- **时间轴**
- **分镜脚本**
- **一致性要求：服装100%与参考图一致，面料纹理100%正确**

**关键要求：**
- 景别要多样化：特写、近景、中景、远景都要有
- 品牌大片标准，不是电商流水线

### 模块5：分镜制作模块

**功能：** 根据选定的脚本，制作两张15秒对应的九宫格分镜图

**输入：** 用户选定的脚本

**输出：** 两张九宫格分镜图的详细描述（适配Seedance2.0模型），每张包含：
- 9个格子
- 每个格子包含：景别、摄像机运动、镜头、焦距、光圈、画面内容、时间轴、转场方式
- 服装100%与参考图一致
- 面料纹理100%正确
- 中文提示词（适配Seedance2.0）

**关键要求：**
- 每张分镜图对应15秒视频
- 景别要多样化：特写、近景、中景、远景都要有
- 所有画面描述必须确保服装与参考图100%一致
- 面料纹理细节必须正确

## 一致性要求

- **服装一致性：** 服装100%与参考图一致
- **面料一致性：** 面料纹理细节100%正确

## 设计原则

1. **线性流程：** 每个步骤都依赖于前一个步骤的结果，确保流程的连贯性
2. **用户参与：** 每个模块都需要用户确认才能继续，确保输出符合用户需求
3. **专业输出：** 以顶尖TVC创意大师的视角进行创作，确保专业性
4. **模型适配：** 所有提示词都针对GPT-image2（英文）和Seedance2.0（中文）模型进行优化
5. **品牌大片标准：** 不是电商流水线，要有故事性、艺术感、电影质感
6. **一致性优先：** 服装和面料与参考图100%一致是最高优先级
