# TVC广告视频创意生成器 - 设计文档

## 基本信息

- **Skill名称：** `tvc-video-creator`
- **创建日期：** 2026-05-16
- **版本：** 1.0
- **状态：** 设计完成

## 核心定位

面向面料/服装行业的TVC广告视频创意生成器，专注于20秒竖屏（9:16）短视频的创意构思、脚本撰写和分镜制作。

## 输入输出

### 输入素材
- 服装图（用户已用面料图生成）
- 模特图

### 输出成果
两张10秒对应的九宫格分镜图（纯中文）

### 技术适配
- 生图模型：GPT-image2
- 视频模型：Seedance2.0

## 流程设计

```
输入素材（服装图+模特图）
    ↓
模块1：创意构思（3组方向）→ 用户选择1组
    ↓
模块2：视觉风格（3种选择）→ 用户选择1种
    ↓
模块3：概念图生成（3张描述）→ 用户确认调性
    ↓
模块4：脚本撰写（3版脚本）→ 用户选择1版
    ↓
模块5：分镜制作（两张10秒九宫格分镜图）
    ↓
最终交付
```

## 模块详细设计

### 模块1：创意构思模块

**功能：** 以顶尖TVC创意大师的视角，根据服装特点和模特气质，构思整体创意

**输入：** 服装图 + 模特图

**输出：** 3组创意方向，每组包含：
- 创意主题/概念
- 核心视觉表现
- 情感调性
- 目标受众感受

**示例输出格式：**
```
创意方向A：
主题：XXX
视觉表现：XXX
情感调性：XXX

创意方向B：
...

创意方向C：
...
```

### 模块2：视觉风格模块

**功能：** 根据选定的创意方向，提供3种不同的视觉风格供用户选择

**输入：** 用户选定的创意方向

**输出：** 3种视觉风格选择，每种包含：
- 风格名称（如：极简主义、电影质感、梦幻柔光等）
- 画面调性描述
- 色彩倾向
- 光影特点
- 参考风格关键词

**示例输出格式：**
```
风格A：极简主义
画面调性：干净、留白、高级感
色彩倾向：低饱和度、莫兰迪色系
光影特点：柔和自然光

风格B：电影质感
...

风格C：梦幻柔光
...
```

### 模块3：概念图生成模块

**功能：** 根据选定的创意方向和视觉风格，生成3张概念图的描述（适配GPT-image2模型）

**输入：** 选定的创意方向 + 视觉风格

**输出：** 3张概念图的详细描述，每张包含：
- 画面构图描述
- 色彩/光影描述
- 主体（模特+服装）呈现方式
- 背景/环境描述
- 适配GPT-image2的提示词

**示例输出格式：**
```
概念图A：
画面描述：XXX
GPT-image2提示词：[针对该模型优化的提示词]

概念图B：
...

概念图C：
...
```

### 模块4：脚本撰写模块

**功能：** 根据选定的创意方向、视觉风格和确认的概念图调性，撰写视频脚本

**输入：** 选定的创意方向 + 视觉风格 + 概念图调性 + 模特气质 + 服装风格

**输出：** 3版脚本供用户挑选，每版包含：
- 视频总时长：20秒（两段各10秒）
- **视觉调性**（与模特气质和服装风格相参照）
- **景别**（远景/全景/中景/近景/特写）
- **摄像机运动**
- **镜头类型**
- **焦距**
- **光圈**
- **画面内容描述**
- **时间轴**
- **分镜脚本**
- **一致性要求：模特100%与参考图一致，服装100%与参考图一致**

**示例输出格式：**
```
脚本A（20秒）：
视觉调性：优雅、高级、XXX风格（与模特气质和服装风格相匹配）

第一段（0-10秒）：
  时间轴：0-3秒
  景别：全景
  摄像机：XXX
  镜头：XXX
  焦距：XXX
  光圈：XXX
  画面内容：XXX
  
  时间轴：3-6秒
  景别：中景
  ...

脚本B（20秒）：
...

脚本C（20秒）：
...
```

### 模块5：分镜制作模块

**功能：** 根据选定的脚本，制作两张10秒对应的九宫格分镜图

**输入：** 用户选定的脚本

**输出：** 两张九宫格分镜图的详细描述（适配Seedance2.0模型），每张包含：
- 9个格子
- 每个格子包含：景别、摄像机运动、镜头、焦距、光圈、画面内容、时间轴、转场方式
- 模特100%与参考图一致
- 服装100%与参考图一致

**示例输出格式：**
```
第一段分镜（0-10秒）：
格子1：
  时间轴：0-1秒
  景别：全景
  摄像机：XXX
  镜头：XXX
  焦距：XXX
  光圈：XXX
  画面内容：XXX
  转场：XXX

格子2：
  ...

格子3：
  ...

...

格子9：
  ...

第二段分镜（10-20秒）：
格子1：
  ...

格子2：
  ...

...

格子9：
  ...
```

**关键要求：**
- 每张分镜图对应10秒视频
- 所有画面描述必须确保模特和服装与参考图100%一致
- 提示词适配Seedance2.0模型

## 一致性要求

- **模特一致性：** 模特100%与参考图一致
- **服装一致性：** 服装100%与参考图一致

## 设计原则

1. **线性流程：** 每个步骤都依赖于前一个步骤的结果，确保流程的连贯性
2. **用户参与：** 每个模块都需要用户确认才能继续，确保输出符合用户需求
3. **专业输出：** 以顶尖TVC创意大师的视角进行创作，确保专业性
4. **模型适配：** 所有提示词都针对GPT-image2和Seedance2.0模型进行优化
