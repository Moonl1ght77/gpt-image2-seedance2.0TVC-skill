# TVC Video Creator Skill

This skill helps generate creative content for 20-second vertical TVC (Television Commercial) advertisements.

## Usage

Invoke this skill when you need to create:
- Video ad concepts and scripts
- Storyboards for short-form vertical content
- Creative briefs for product advertising

## Example Prompts

- "Create a TVC script for a skincare product"
- "Generate a 20-second vertical video concept for a food delivery app"
- "Develop a storyboard for a fashion brand advertisement"

## Output

The skill generates structured creative documents with concepts, timed scripts, and visual storyboards optimized for vertical (9:16) format.
