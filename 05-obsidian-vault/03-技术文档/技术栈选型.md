# 技术栈选型

> 2026-04-28 确定

## 前端（小程序）
- **方案**：微信原生小程序
- **理由**：性能最好，无额外框架开销，适合展示型小程序
- **基础库版本**：3.6.0

## 后端
- **方案**：微信云开发
- **理由**：免服务器、免备案、免运维，展示型项目免费额度够用
- **启用时机**：先做前端框架，后期接入云开发处理询价表单

## 数据库
- **方案**：微信云数据库（云开发自带）
- **数据集合规划**：
  - `fabrics` — 面料数据（名称、参数、色号、图片）
  - `inquiries` — 询价表单提交记录
  - `visitors` — 访问统计（可选）

## 部署
- **方式**：微信小程序平台直接发布
- **测试**：微信开发者工具模拟器 + 真机预览
- **审核**：提交微信审核（需营业执照等资质）

## 设计规范
- **配色**：暖米白(#F5F0EB) + 浅木棕(#8B7355) + 莫兰迪绿(#B8C5B8)
- **圆角**：sm 12rpx / md 20rpx / lg 32rpx
- **阴影**：rgba(139,115,85,0.08) 轻 / rgba(139,115,85,0.12) 浮
- **动画**：滚动触发（IntersectionObserver）+ CSS transition/animation
- **字体**：系统默认（PingFang SC）
