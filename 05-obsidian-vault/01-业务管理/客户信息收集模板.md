# 客户信息收集模板

> 新客户接入时复制此模板，填写后保存到 `02-客户档案/` 文件夹
> 文件名格式：`客户-公司名.md`

**必须在文件最顶部加上以下元数据（复制粘贴后修改）：**
```yaml
---
类型: 客户档案
客户名称: （公司名）
行业: （行业）
状态: 需求收集中
阶段: 需求收集
启动日期: （YYYY-MM-DD）
合作方式: 待定
套餐: 待定
---
```
**状态可选值**：需求收集中 / 设计中 / 开发中 / 测试中 / 已上线 / 维护中 / 已结束
**阶段可选值**：需求收集 / 竞品调研 / 方案设计 / 开发 / 测试与交付 / 维护
