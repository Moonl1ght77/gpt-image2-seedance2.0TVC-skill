# SOP-20260513：联系页转化优化 + 营业执照页 + 全局启动/跳转动画

## 背景与痛点

1. 详情页底栏有两个按钮（"立即询价"+"联系客服"），用户选择困难，转化分散
2. 首页底部"开启合作"CTA 样式突兀，且无法体现合规性
3. 没有品牌感的启动动画，冷启动体验差
4. 页面切换无过渡感，体验割裂

---

## 优化前后对比

| 项目 | 优化前 | 优化后 |
|------|--------|--------|
| 详情页底栏 | 两个按钮（询价+客服） | 单个全宽"立即询价"金色按钮 |
| 首页底部 | 灰色"开启合作"CTA卡片 | 电子营业执照信息栏 + 品牌logo |
| 启动体验 | 直接进首页，无过渡 | Splash页 logo渐入渐出动画 |

| 营业执照 | 无 | 专属页面，米白主题+返回导航 |

## 核心可复用代码

### 1. 全局 loading-mask 组件

```js
// components/loading-mask/loading-mask.js
Component({
  data: { visible: false },
  methods: {
    show(duration = 400) {
      this.setData({ visible: true })
      setTimeout(() => this.setData({ visible: false }), duration)
    }
  }
})
```

```css

.lm-overlay {

@keyframes logoReveal {
  0%   { opacity: 0.12; }
```

```xml
<!-- app.json 全局注册 -->
"usingComponents": {
  "loading-mask": "/components/loading-mask/loading-mask"
}

<!-- 每个页面 WXML 第一个子元素 -->
<loading-mask id="lmask" />
```

```js
// 每个页面 onShow 触发（首页首次跳过）

  const mask = this.selectComponent('#lmask')
  if (mask) mask.show()
}
```

## 4. 面料墙（横向 Bento Grid）

### 背景
首页"面料工艺"下方需要一个视觉冲击力强的图片展示区，展示面料效果图，增强品牌质感。

### 核心可复用结构

```xml
<!-- scroll-view 必须套一层内层 view，不能直接加 display:flex -->
<scroll-view scroll-x="{{true}}" enhanced="{{true}}" show-scrollbar="{{false}}">
  <view class="fw-row">
    <!-- 宽大格（全高） -->
    <view class="fw-item fw-w340">
      <image class="fw-img" src="{{imgs[0]}}" mode="aspectFill" lazy-load />
    </view>
    <!-- 纵向双格 -->
    <view class="fw-column-group fw-w200">
      <view class="fw-item-half"><image class="fw-img" src="{{imgs[1]}}" mode="aspectFill" lazy-load /></view>
      <view class="fw-item-half"><image class="fw-img" src="{{imgs[2]}}" mode="aspectFill" lazy-load /></view>
    </view>
    <!-- 重复以上两种格子 -->
  </view>
</scroll-view>
```

```css
/* scroll-view 只设宽度，flex 放在内层 */
.scroll-view { width: 100%; }
.fw-row { display: inline-flex; align-items: stretch; height: 440rpx; gap: 12rpx; padding: 0 32rpx; }
.fw-item { flex-shrink: 0; height: 100%; border-radius: 16rpx; overflow: hidden; }
.fw-column-group { display: flex; flex-direction: column; flex-shrink: 0; gap: 12rpx; height: 100%; }
.fw-item-half { flex: 1; border-radius: 14rpx; overflow: hidden; }
.fw-w200 { width: 200rpx; }
.fw-w340 { width: 340rpx; }
```

### 踩坑
- **scroll-view 不支持直接 display:flex**：必须在内层 `<view>` 上加 flex，否则只显示第一格，其余消失
- **图片必须传云存储**：8张图合计约10MB，超包体积限制
