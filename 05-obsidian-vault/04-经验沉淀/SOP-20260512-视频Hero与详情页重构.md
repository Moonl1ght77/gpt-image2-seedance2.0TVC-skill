# SOP：首页视频 Hero + 面料详情页重构

## 需求背景

- 首页图片轮播改为全屏 TVC 视频，去除所有文字层，保留暗角效果
- 视频结尾展示 Logo 动画（导航栏同款图片），循环播放
- 面料详情页去掉顶部轮播大图，改为纯信息卡片置顶 + 默认色卡视角
- 产品视角内容去掉 emoji 图标，改为极简高级排版
- 修复搜索图标跳转导致 tabBar 消失的导航 bug

## 核心可复用代码

### 1. 视频最后 N 秒显示 Logo 动画

```js
// index.js
onVideoTimeUpdate(e) {
  const { currentTime, duration } = e.detail
  if (!duration) return
  const nearEnd = currentTime >= duration - 5  // 最后5秒
  const looped = currentTime < 1 && this.data.showLogoAnim
  if (nearEnd && !this.data.showLogoAnim) {
    this.setData({ showLogoAnim: true })
  } else if (looped) {
    this.setData({ showLogoAnim: false })
  }
},
```

### 4. tabBar 消失 Bug 修复（关键坑）

**原因**：用 `navigateTo` 打开 tab 页，tab 页会以普通页方式加载，`getTabBar()` 返回 null，tabBar 不显示。  
**规则**：所有跳转到 tabBar 页面的导航，必须用 `switchTab`，不能用 `navigateTo`。

```js
// ❌ 错误
wx.navigateTo({ url: '/pages/category/category' })

// ✅ 正确
wx.switchTab({ url: '/pages/category/category' })
```

> ⚠️ `switchTab` 不支持 URL 传参，如需传状态可用 `app.globalData` 中转。
