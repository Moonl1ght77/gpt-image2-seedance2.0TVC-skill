# SOP-20260514 · 搜索页 + 询价通知云函数

---

## 背景痛点

1. **搜索页**：分类页原有搜索框只是 UI，无过滤逻辑；需要独立搜索页支持关键词搜索 + 历史记录 + 最近浏览
2. **询价通知**：客户提交询价后商家完全不知道，只能自己登云控制台翻数据，体验极差
3. **scroll-view 空白 bug**：微信小程序 scroll-view 在 flex 布局中必须绑定明确高度数值，否则内容不渲染

---

## 优化前后对比

| 项目 | 优化前 | 优化后 |
|------|--------|--------|
| 搜索 | 分类页内联搜索框，无过滤逻辑 | 独立搜索页，支持关键词过滤/历史/浏览 |
| 询价通知 | 无任何通知机制 | 云函数自动推送订阅消息到商家微信 |
| scroll-view | 内容空白（min-height+flex） | JS 计算高度绑定到 style |

## 核心可复用代码

### 1. scroll-view 正确高度写法（微信小程序）

```js
// onLoad 计算可用高度
onLoad() {
  const navH = app.globalData.navbarHeight || 44
  const sys = wx.getSystemInfoSync()
  this.setData({ scrollHeight: sys.windowHeight - navH - 72 })
}
```

```xml
<!-- WXML 绑定高度 -->
<scroll-view scroll-y style="height: {{scrollHeight}}px;">
  <!-- 内容 -->
</scroll-view>
```

```css
/* WXSS 父容器必须用 height 不用 min-height */
.page-wrap {
  height: 100vh;
  display: flex;
  flex-direction: column;
  overflow: hidden;
}
.my-scroll {
  flex: 1;
  overflow: hidden;
}
```

### 2. 搜索历史管理（localStorage）

```js
const HISTORY_KEY = 'searchHistory'
const MAX_HISTORY = 8

// 保存历史
_saveHistory(keyword) {
  let list = wx.getStorageSync(HISTORY_KEY) || []
  list = list.filter(k => k !== keyword)
  list.unshift(keyword)
  if (list.length > MAX_HISTORY) list = list.slice(0, MAX_HISTORY)
  wx.setStorageSync(HISTORY_KEY, list)
  this.setData({ searchHistory: list })
},

// 清除历史
clearHistory() {
  wx.removeStorageSync(HISTORY_KEY)
  this.setData({ searchHistory: [] })
}
```

### 4. 询价通知云函数（notifyInquiry/index.js）

```js
const cloud = require('wx-server-sdk')
cloud.init({ env: cloud.DYNAMIC_CURRENT_ENV })
const db = cloud.database()

exports.main = async (event) => {
  const { name, phone, fabric, submitTime } = event
  const res = await db.collection('merchant_config').doc('main').get()
  const merchantOpenid = res.data.openid

  await cloud.openapi.subscribeMessage.send({
    touser: merchantOpenid,
    templateId: 'YOUR_TEMPLATE_ID',
    page: 'pages/contact/contact',
    data: {
      thing1: { value: fabric },
      phone_number2: { value: phone },
      name3: { value: name },
      time4: { value: submitTime }
    }
  })
  return { success: true }
}
```

> ⚠️ config.json 需加 `"permissions": { "openapi": ["subscribeMessage.send"] }`

### 5. 商家订阅按钮（contact.js）

```js
subscribeMerchantNotify() {
  wx.requestSubscribeMessage({
    tmplIds: ['YOUR_TEMPLATE_ID'],
    success: (res) => {
      if (res['YOUR_TEMPLATE_ID'] === 'accept') {
        wx.cloud.callFunction({ name: 'saveMerchantOpenid' }).then(() => {
          wx.showToast({ title: '通知已开启 ✓', icon: 'success' })
        })
      }
    }
  })
}
```
